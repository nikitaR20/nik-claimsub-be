To Load the files, follow the process-
1. Store all the csv files in a folder
2. In the SQL folder, modify path = full path of csv files
3. Open PGAdmin -> Go to PSQL tool workspace -> Enter the database credentials to which data is to be loaded
4. Connect and open PSQL tool
5. Once in the terminal, copy-paste the sql scripts stored in the SQL folder in the following sequence:
- providers
- risk ratings
- claims
- claim_riders
- claim_claim_rider
- customer_segments
- provider_customer_segments
